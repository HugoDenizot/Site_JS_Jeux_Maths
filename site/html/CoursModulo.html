Principe des modulos :


Les modulos sont présents dans notre quotidien sans qu'on le sache


Exemple avec le système des heures :

Une journée dure 24 heures, après 23h on retombe à 00h.
Donc on peut dire qu'une journée est modulable par 24, on écrit 24 mod 24 = 0

Toutefois nous n'obtenons pas tout le temps 0.
parfois on obtient un reste : X mod Y = reste


Exemple :

9 mod 4 = 1
On enlève autant de fois que possible 4 dans 9 (ici 2 fois) (4 × 2) = 8
9 - 8 =1 , Donc il nous reste 1.
13 mod 5 = 3
On enlève 5 dans 13 deux fois et il nous reste 3.


Propriétés :

- Si on a A = B mod N

Alors on peut avoir :

            - A × k = B × k mod N
            - A + k = B + k mod N

On fait comme pour une équation (attention à bien avoir le A = B mod N)

- Si A = B mod N alors B = A mod N
- Si on a A = B mod N et C = D mod N

Alors :
            - A + C = B + D mod N
             - A × C = B × D mod N

On retrouve la propriété du début mais elle est un peu plus poussée.


Petite astuce :

Pour trouver plus rapidement le résultat d'un calcul modulaire vous pouvez (si le cas le permet) rechercher si le nombre est divisible par un petit nombre :


Exemple :

1546 mod 3


Si on additionne tous les chiffres de 1546 on obtient : 1 + 5 + 4 + 6 = 16
16 n’est pas divisible par 3, mais 15 oui donc on a 1546 mod 3 = 1
( 15 mod 3 = 0 donc 16 mod 3 = 1)


Pour 5 c’est facile . Si le nombre se termine par 0 ou 5 alors la réponse est 0.
Pour 2 si le nombre est pair ou impair. Si le nombre est pair le résultat est 0, si le nombre est impair le résultat est 1
Pour 7 là c’est déjà plus compliqué :

On a un nombre de la forme suivante A + B, avec :
A qui correspond au nombres des dizaines et au-dessus
( 17526, A = 1752 et B = 6)
B chiffre des unités.
A qui correspond au nombre des dizaines
On va faire avec : A + 5 × B = C
Si « C » est trop grand pour savoir s’il est divisible par 7, alors on recommence avec C
Si on arrive à trouver facilement que « C » est divisible par 7, alors c’est bon.
( C = 21 par exemple)

Exemple : 17 381

=> A = 1738 et B = 1
1738 + 5 × 1 = 1743

=> A = 174 et B = 3
174 + 5 × 3 = 189

=> A = 18 et B = 9
18 + 5 × 9 = 63

=> A = 6 et B = 3
6 + 5 × 3 = 21

=> 21 est divisible par 7 (3×7)
Donc le résultat est correct.


Cette technique est un peu longue mais permet de faire le calcul sur des gros nombres.
Il faut juste faire attention à ne pas aller trop vite au risque de se tromper dans les A et B.
