Produits de puissances d’un même nombre :


Exemple :

24 × 25 = (2×2×2×2) × (2×2×2×2×2) = 29
52 × 53 = 5(2+3) = 55

Propriété : Pour tous nombres relatifs a, n et p, on a :

an × ap = a(n+p)

Puissance de 0 :

Exemple :

20=1
128÷90=128

Propriété : Pour tout nombre relatif a non nul, on a :

a0=1

Puissance de puissance :

Exemple :

(23)2 = (23) × (23) = 2(3+3) = 26
(86)3 = 8(6+6+6) = 818

Propriété : Pour tous nombres relatifs a, n et p, on a :

(an)p = a(n×p)

Puissance d’exposant négatif :

Exemple :

23 × (1 ÷ (23)) = 2×2×2 × (1 ÷ (2×2×2))
= ((2×2×2) ÷ (2×2×2)) = 1
On écrit (23) * 2(-3) parce que 2(-3) est égal à 1÷(23)

Exemple complémentaire :

3(-2) = 1÷32 = 1÷9
5(-3) = 1÷53 = 1÷125

Propriété : Pour tous entiers relatifs a et n, on a :

a(-n) = 1÷an

Quotient de deux puissances d’un même nombre :

Exemple :

25÷22 = (2×2×2×2×2÷(2×2)) = 2×2×2 = 23
34÷36 = (3×3×3×3÷(3×3×3×3×3×3)) = 1÷(3×3) = 1÷(32) = 3(-2)
43÷41 = (4×4×4)÷(4)=42

Propriété : Pour tous nombres relatifs a, n et p, on a :

an÷ap = a(n-p)

Puissance d’un produit :

Exemple :

(2×3)4 = (2×3) × (2×3) × (2×3) × (2×3) = (2×2×2×2) × (3×3×3×3) = 24 × 34
43 × 73 = (4×7)3 = 283

Propriété : On a A et B deux nombres non nuls et n un entier :

(A × B)n = An × Bn

Puissance d’un quotient :

Exemple :

(2÷5)3 = (2÷5) × (2÷5) × (2÷5) = (2×2×2)÷(5×5×5) = (23)÷(53)
367÷37= (36÷3)7 = 127

Propriété : On a A et B deux nombres non nuls et n un entier :

(A÷B)n = (An÷Bn)

103 = 1 000
10(-2) = 1÷102 = 1÷100 = 0.01

10n = 10×10×…×10 = 100…0
